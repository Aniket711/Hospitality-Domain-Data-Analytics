# Hospitality-Domain-Data-Analytics

Project Overview

This project performs Exploratory Data Analysis (EDA) on hotel booking datasets to derive business insights related to revenue, occupancy, room performance, and booking behavior.
The analysis includes data cleaning, outlier detection, missing value treatment, aggregation, and feature engineering to calculate key performance metrics such as occupancy percentage.

Tech Stack

Python
Pandas
Matplotlib
Jupyter Notebook

Dataset Description

The project uses the following datasets:
1. fact_bookings.csv
2. fact_aggregated_bookings.csv
3. dim_hotels.csv
4. dim_rooms.csv
5. dim_date.csv
These datasets contain booking details, hotel information, room categories, and aggregated booking metrics.

Key Insights Derived

Booking platform distribution,
Revenue distribution patterns,
Room category performance comparison,
Average occupancy rate by room class,
Identification of capacity inconsistencies in raw data

Business Metrics Calculated

Total successful bookings per property,
Revenue statistics (mean, median, min, max),
Occupancy percentage,
